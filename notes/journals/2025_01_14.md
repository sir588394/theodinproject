- [[html_tag]] 
  date:: 2025-01-14
  link:: https://www.geeksforgeeks.org/html-introduction/
  tags:: html, web, IoT, html_tag
  type:: [[fleeting]]
	- [[html_tag]]s are keywords delimiting [[html_elements]] telling the browser what content to expect.
	- [[html_tag]]s keywords are enclosed by "<*keyword*>" at the start and "</*keyword*>" at the end if needed.
- [[html_element]]
  date:: 2025-01-14
  link:: https://www.geeksforgeeks.org/html-introduction/
  tags:: html, web, IoT, html_element
  type:: [[fleeting]]
	- [[html_element]]s are structures of the following form
	  ```html
	  <keyword> content if any </keyword>
	  ```
- [[html_document_type]] declaration
  date:: 2025-01-14
  link:: https://www.geeksforgeeks.org/html-introduction/
  tags:: html, web, IoT, html_document_type
  type:: [[fleeting]]
	- [[html]] document start with a [[html_document_type]] declaration, which is not a [[html_tag]].
	- The declaration for [[html]]5 [[html_document_type]] is
	  ```html
	  <!DOCTYPE html>
	  ```
